## Tutorial de instalação

Clone o projeto

```term

git clone git@github.com:moraes3000/ignite-First-web-application-with-ReactJS.git

```

Instale as dependências

```term

yarn

```

Rode o projeto

```term

yarn dev

```

Acesse o seguinte endereço

```term

http://127.0.0.1:8080/

```
